Different Types of Viruses

There are different types of malware:

  • Worms: A computer program that replicates itself at
    a fast pace. It is self-contained and doesn't need to be part of
    another program to multiple itself.
  • Trojan Horse: A destructive program that is disguised in a normal
    software program. The program cannot replicate itself like other viruses.
  • Spyware: A software that gathers information about a person/group
    with their consent. The same process with the user's consent is
    done through cookies.
